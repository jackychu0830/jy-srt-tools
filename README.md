# 剪映字幕工具箱

## 主要功能：

1. 草稿字幕簡體轉繁體
2. 字幕匯出成 SRT 檔 及 TXT 文字檔
3. 載入既有的 SRT 檔
4. 字幕編輯修改
5. 刪除影片字幕
